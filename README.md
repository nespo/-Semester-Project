# StudyBuddy Planner

A simple study task planner that helps students track their study tasks, priorities and basic progress.

## Features

- [x] Add, edit and delete study tasks
- [x] Mark tasks as completed / pending
- [x] Set priority (1–5) and optional due date as text
- [x] Stats screen with totals, completed, pending and high‑priority tasks
- [x] Dark mode via system theme (Material 3)

## Tech Stack

- Android (Kotlin, Jetpack Compose)
- Architecture: MVVM (ViewModel + Repository + Room)
- Storage: Room
- Navigation: Compose Navigation
- Testing: JUnit (unit test for input validation)

## How to Run

1. **Install Android Studio (Koala or newer)** on your Mac and make sure you have an Android SDK (API 26+).  
2. Download and unzip this project, or clone it from your own GitHub repository after you upload it.
3. Open Android Studio → **Open** → select the `StudyBuddyPlanner` folder.
4. If Android Studio asks to **Sync Gradle**, accept and wait until it finishes.
5. Create an Android emulator (API 26+), or connect a real Android device with USB debugging enabled.
6. Press the green **Run** ▶ button and choose your device/emulator.

The app should install and start with the **Task list** screen.
